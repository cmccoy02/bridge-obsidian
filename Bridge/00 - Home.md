## The one-liner

Bridge is a technical debt intelligence platform. It scans codebases, scores debt across six dimensions, surfaces actionable fixes, and integrates into every workflow an engineering team already uses — from the terminal to Claude Code to GitHub PRs.

The bet: as AI-generated code floods engineering teams, visibility into codebase health becomes existential. Bridge is the system that keeps both humans and agents working with integrity.
